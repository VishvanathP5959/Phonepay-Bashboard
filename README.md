# Phonepay-Bashboard
Recommended Structure and Order
1. Project Title / Headline
PhonePe Transaction Insights Dashboard.

2.Short Description / Purpose
An interactive Power BI dashboard built using PhonePe transaction data to analyze payment trends, transaction values, user behavior, and service performance. The dashboard helps businesses monitor KPIs and gain actionable insights for data-driven decision-making.

3.Tech Stack
📊 Power BI

📄 CSV
🗄️ Data Modeling
📉 Data Visualization
🐙 Git & GitHub
